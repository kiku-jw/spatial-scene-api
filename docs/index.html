What this page does

This is the browser-only version of the parked Spatial Scene prototype. It uses a deterministic heuristic depth map and tiled canvas warping. It is not generative video and does not send the image anywhere.

Export uses the browser encoder. Some browsers produce MP4, others produce WebM.
